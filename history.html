History and Evolution of Internet

Let's dive in to a brief history and evolution of browsers and web technologies

Author: Akhil Eaga Follow

Date of publication: 12-April-2021

0. Getting started

What is a web browser?

A web browser is a software application for retrieving, presenting and traversing information resources on the World Wide Web. - Wikipedia

As understood from the above, we all know that the fundamental role of the browser is to connect us with everything on the web. Our browser allows to search the web, find videos, uplod multimedia, play game, use web applications, and much much more. Technically speaking a browser is a software application that is capable to retrieving webpages, images, videos etc from the server and render it on the clients' computer.

There are many popular browsers like Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari, Opera etc.. Each browser respond diffently to webpages. This difference in the browsers' response led to cross-browser testing. Let's take a look at the history of these major browsers and their backend to understand what it takes to design and develop a cross-compatible website.

1. Timeline of Browsers

There some browsers which were used in the past but are no longer in use today or some of those have evolved to a great extent.

1990

Tim Berners Lee, the inventor of the World Wide Web, created the world's first browser named WorldWideWeb.

1992

A text-based browser named Lynx that could not display images or any kind of graphic was created.

1993

The world's first most popular browser capable of embedding images in text was created. It is the Mosaic.

1994

Netscape Navigator - a significant improvement to Mosaic was made.

1995

Microsoft launched it's first browser - Internet Explorer.

1996

This year marked the start of browser wars (arguably). As Internet Explorer forged ahead with new features as compared to Netscape Navigator 3. This is the year when Opera went public after starting as research project in 1994.

2003

Instead of using Navigator, Apple released it's own Safari browser specific to their Macintosh computers.

2004

As Netscape Navigator faded into the dark, Mozilla launched Firefox.

2007

Apple launched their mobile version of Safari which till date continues to lead in the the iOS market.

2008

Google launched their browser Google Chrome which quickly took over the browser market.

2011

To focus on the fast-growing mobile browser market, Opera launched their mobile version called Opera Mini.

2015

Microsoft launched Edge to combat Google Chrome

2020

In August of 2020, Microsoft announced that Internet Explorer will no longer be supported from 17 August, 2021, after the final update in November 2020

2. History of Web technologies: HTML/CSS/JavaScript

A brief history of HTML: 1993 - Present

Tim Berners Lee wrote the first version of HTML in 1993. Many versions of HTML were developed since then. HTML 4.01 became an official standard in the December of 1999, and was the most widely used version of HTML throughout the 2000’s.

An XML version of HTML was created which was named XHTML. As a standard markup language XML was used to create several other markup languages. The content created in any of these markup languages can be easily across different applications because all these languages are written from XML. W3C created an XML version of HTML, called XHTML. In 2000, XHTML became an official standard which was updated 2 years later in 2002. While HTML is similar to XHTML, it has less restrictions.

As HTML 4.01 was the standard in 2000’s much of the Web was either built on HTML 4.01 or XHTML 1.0. During the time of 2010-2011, W3C collaborated with another organization called WHATWG, and worked on developing a newer version of HTML. Even though it was in draft specification, HTML5 was widely supported by browsers and other devices which are capable of accessing the web, and paved the way into the future. The new version was named HTML5 and became an official standard in 2014 and has been the standard since then.

As new versions emerge, HTML started to support more and more types of content.

During the early years of HTML, when web designers wanted to add some unsupported content, they had to use a work around and make use of proprietary, but non-standard software like the Adobe Flash. Many a times, browser plug-ins installation was required to access those content and, in some cases, certain users with devices like iPhone and iPads do not have support to Adobe Flash.

Hence HTML5 added a bunch of new features without relying on that non-standard proprietary software and enabled to do more with just the new version of HTML.

A brief history of CSS: 1996 - Present

Hakon Wium Lie was the first to propose CSS on October 10 of 1994. Lie was working with Tim Berners Lee at CERN back at that time. Several other style sheet languages for the web were proposed at that time. Bert Bos’s proposal was an influential one and he became the co-author of CSS1 and has been regarded as the co-creator of CSS in 1996.

Issues that were not addressed in CSS level 1, were tackled by the CSS Working Group resulting in the creation of CSS level 2 on 4th of November in 1997. CSS level 2 was published as a W3C recommendation on 12th May 1998. Development of CSS level 3 started back in 1998 and is the current standard in CSS.

Each version of CSS builds up on the previous version and are named sequentially to denote the versions. Some of the notable advancements in each version are discussed below:

CSS 1

Font properties, color & backgrounds of elements, text attributes, alignment of tables and images, margin, border, padding, unique identification and generic classification of groups of attributes.

CSS 2

This version of CSS brought new features like absolute, relative and fixed positioning of elements, z-index, mediatypes and font shadows.

CSS 2.1

This version of CSS fixed the errors that were present in CSS2 and removed poorly supported or not completely interoperable features and adds already implemented browser extensions to the official specification. CSS 2.1 became official standard on 7th June of 2011

CSS 3

Unlike the second version of CSS which is a single large specification, CSS3 is broken down into several documents called modules. Each module added or extended the functionality defined in CSS2 while preserving backward compatibility.

CSS 4

There is no one single, integrated CSS4 specification, because it has been split in to different modules and are levelled independently.

CSS Working Group sometimes publishes what are called as “Snapshot” which are essentially a collection of stable enough modules to be implemented by browsers.

A brief history of JavaScript: 1996 - Present

JavaScript is ubiquitous (*almost) and is one of the most commonly used programming languages in the world. It comes as no surprise that that it has been ranked “the most commonly used” programming language for 8 years in a row. As of 2019, over 65% of developers employed JavaScript. The rise of JS is sometimes referred synonymously to the rise of internet itself.

JS was created out of necessity and is currently used to build more than 95% of the websites including the giants like Facebook, YouTube and eBay. JS adds a functionality and interactivity to websites and webpages.

During the mid-1990s, key players like Netscape and Microsoft were in the middle of browser war with both the browsers competing neck to neck. In 1995, a programmer from Netscape, Brandan Eich developed a new scripting language in just 10 days (Phew!!!). That first version was named as “Mocha” which later changed to LiveScript and now it is being referred as JavaScript.

Due to the rapid growth of JavaScript, Netscape handled the job of creating a language specification to European Computers Manufacturers Association (ECMA) in 1997.

2005 marks a great year for JavaScript because of a paper released by Jesse James Garett who introduced Ajax, which is a revolutionary suite that made use of JS. Ajax makes the webpages looks and feel more like native apps thus improving the user experience. Although at that time, JS faced many challenges, like for example, being verbose to achieve simple tasks, but the community responded by creating popular JS frameworks and libraries like jQuery, Dojo and Mootools.
