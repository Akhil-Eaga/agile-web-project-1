2. CSS Examples

CSS can be added to a html document in 3 different ways. They are:


  1. Inline CSS
  2. Internal CSS
  3. External CSS

The following code window shows an example of the inline css. The code for other styling methods are hard to show in this page. Hence the code will be shown somewhat differently.


Example 1: Inline CSS

Inline CSS rules added using the style attribute within each element like it is shown below:

<p style = "background-color: cadetblue; color: white; font-style: italic; padding: 5px; border: 2px solid black; border-radius: 3px;"> This is a <p> with inline styling added to it. This styling makes it look good.</p>

This is a <p> with inline styling added to it. This styling makes it look good.

Example 2: Internal CSS

Internal CSS rules are to be written inside the <style> tag which is supposed to be placed in the <head> tag of each html file.


<style> /* html element is used as selector */ p { /* This is a comment in CSS */ color: green; background-color: black; font-size: 1rem; padding: 1rem; } </style>

This is styled paragraph element


Example 3: External CSS

External CSS rules are written in a totally different file and the <link> tag is used to connect the external style sheet and the html files to apply the styles written in the external style sheet


/* css code in file styles.css */ /* id is used as selector */ #special-button { color: white; background-color: black; border-radius: 10px; padding: 0.5rem 1rem 0.5rem 1rem; border-style: none; border: 3px solid black; font-size: 1rem; font-family: "Ubuntu Mono"; transition: background-color 0.5s; } /* pseudo class hover is used to change css properties on hovering over the element */ #special-button:hover { color: black; background-color: white; } -------------------------------------------- <!-- html code in index.html file --> <html> <head> <link rel="stylesheet" href="styles.css"> </head> <body> <button id="special-button">Get in touch</button> </body> </html>



** The above button is created using the styles mentioned in the code pane.
